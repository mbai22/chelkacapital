# Chelka Capital

Site vitrine pour une société d'investissement et courtage.

## Sections

- **Hero** — Pleine hauteur avec background image et overlay
- **Stats** — Chiffres clés (clients, capital, avis)
- **Offres** — 3 formules d'investissement (Discovery, Growth, Premium)
- **Services** — 6 domaines d'expertise financière
- **Témoignages** — Slider automatique avec avis clients
- **Certifications** — Labels et agréments (ACPR, ISO, AMF, PCI)
- **FAQ** — Accordéon de questions fréquentes
- **Blog** — 3 articles avec images
- **Contact** — Formulaire + coordonnées
- **Footer** — Liens et crédits

## Stack

- HTML / CSS / JS vanilla
- Font Awesome 6 (icônes)
- Google Fonts (Inter)
- Images Unsplash
